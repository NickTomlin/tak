Language Reference

Syntax

Literals

42          // integer
3.14        // float
"hello"     // string (supports \n \t \r \" \\)
true false  // booleans
null

Function definitions

fn square ( n -- n ) { dup * }
fn greet  ( str -- ) { "Hello, " swap concat . }

Quotations and arrays

[ word... ] is a callable quotation when it contains words. [ literal... ] is an array of values when it contains only literals. Nested literal brackets create nested arrays.

[ 1 2 3 ]         // array literal
[ dup * ]         // callable quotation
[ "a" "b" "c" ]   // array of strings

Dict literals

{ name: "Alice", age: 30 }

Control flow

5 0 > [ "positive" . ] [ "non-positive" . ] if
10 [ dup 0 > ] [ 1 - ] while
5 [ "tick" . ] times
true [ "yes" . ] when

JavaScript escape hatch

@js(document.querySelector('#app'))
@js(Math.random())

Static Imports (use)

use is a hoisted import. All URLs are fetched before the program runs; results are available synchronously at the call site.

"./lib.js" use
"./lib.js" [ Chart ] use
"./lib.js" [ renderFn as render ] use

The URL must be a string literal. Without bindings, the whole module is pushed. With bindings, each export becomes a word.

Debug words

  • trace — log top value without consuming it
  • debug — emit debug event and log the full stack

Debug Panel API

const interp = new Tak.TakInterpreter({ debug: true });

interp.on('beforeWord', ({ word, stack }) => { });
interp.on('afterWord',  ({ word, stack }) => { });
interp.on('push',       ({ value, stack }) => { });
interp.on('pop',        ({ value, stack }) => { });
interp.on('error',      ({ error, word, stack }) => { });
interp.on('scriptEnd',  ({ stack }) => { });

interp.stack   // read-only snapshot
interp.dict    // Map of all defined words

Ctrl+D toggles the debug overlay when data-debug="true" is set.

CLI

tak script.tak        # run a file
echo "3 4 + ." | tak  # stdin
tak                   # interactive REPL

Install: npm install -g @nicktomlin/tak

DOM words are not available in the CLI.

Programmatic Use

import { TakInterpreter, registerStdlib, registerDomStdlib } from './dist/tak.js';
import { tokenize } from './dist/tak.js';
import { parse } from './dist/tak.js';

const interp = new TakInterpreter({ debug: true });
registerStdlib(interp);
registerDomStdlib(interp);

const tokens = tokenize('5 3 + .');
const program = parse(tokens);
await interp.run(program);
